# Django Tutorial: Polls App

This is my try for the [django tutorial app](https://docs.djangoproject.com/en/5.2/intro/tutorial01/)

This respository contains: 
### 1. `djangotutorial/`
The main folder to run the server from.
To run the server:
```
cd djangotutorial
python manage.py runserver
```

### 2. `django-polls/`
A reusable Python package version of the `polls` app, for running the main application.
